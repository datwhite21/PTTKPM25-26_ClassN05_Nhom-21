# Hệ thống Quản lý Đăng ký Học theo Tín chỉ - Nhóm 21 (Lớp N05)

Dự án môn học **Phân tích & Thiết kế Phần mềm (PTTKPM)** - Học kỳ 2025-2026.

## 📌 Tổng quan dự án

Hệ thống hỗ trợ Sinh viên đăng ký lớp học phần trực tuyến theo tín chỉ, giúp Giảng viên quản lý lịch dạy và điểm số, đồng thời cho phép Giáo vụ (Admin) lập lịch, quản lý danh sách môn học, lớp học phần và điều chỉnh trạng thái cổng đăng ký học tập.

### 🛠️ Công nghệ sử dụng
- **Backend**: Java 17+, Spring Boot 3.x, Spring Data JPA, Hibernate, MySQL 8
- **Frontend**: React (Vite), TypeScript, Axios, Vanilla CSS
- **Thiết kế & Công cụ**: UML 2.0, Mermaid.js, Git & GitHub
